# Shell

You can just use the `hdcms` binary that is normally compiled.
Check out the examples in `main.sh` (or `main.ps1` if you're on
windows)

# Python

## Setup

```bash
$ python -m pip install hdcms
```

## Usage

```bash
$ python main.py
$ python minmax.py
```

for more information (in python):

```python
>>> import hdcms
>>> help(hdcms)
```
